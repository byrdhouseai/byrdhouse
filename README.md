# ByrdHouse - AI Tools Platform

**15+ AI Tools for Creators** - Images, Videos, Music, Chat, TTS, and more.

byrdhouse.app | byrdhouse.ai

## Features

AI Chat / Image Generation / Video Generation / Music Generation / Text to Speech / PDF Creator / Presentation Maker / Background Removal / Logo Maker / Thumbnail Generator / Translator / Code AI / AI Writer

- Pro: $9.99/mo - All 15 tools, unlimited credits
- Enterprise: $29.99/mo - Everything + 5 team seats + API

## Setup

```bash
cp .env.example .env
# Fill in your API keys
npm install
npm run dev
```
